玩家使用从编队、切人到反应触发的实际操作教程

如果你是玩家，这一页才是“怎么用插件”的核心教程。

1. 快速上手流程

01
打开角色编队

输入 /ys open 或使用按键桥的打开 GUI 键。

02
把角色放进 1 到 4 号槽

只有角色物品能放进编队 GUI。

03
切换当前出战槽

使用 /ys player set 1-4 或按键桥默认的 1-4 键。

04
拿对武器

把与当前角色类型匹配的原神武器放到主手。

05
开始战斗

释放 E、Q，或者通过普攻/重击挂元素并触发反应。

2. 角色编队

  • 角色编队 GUI 只放角色，不放武器。
  • 总共有 4 个槽位，当前选中槽位决定你的出战角色。
  • 如果某个槽位没有有效角色，切过去后只会切槽，不会进入角色模式。
  • 若当前槽位角色无效，系统会尽量归位到第一个有效角色槽。

3. 主手武器判定

  • 当前插件不在 GUI 里存武器，而是直接检测主手。
  • 角色模式激活的标准流程是：已选角色存在 + 主手武器类型匹配。
  • 比如刻晴需要单手剑，夜兰需要弓，凝光需要法器。
  • 如果主手武器不匹配，角色技能和角色普攻逻辑不会按完整角色模式运行。

4. 技能怎么放

  • 命令施放：/ys skill e、/ys skill q
  • 按键桥默认绑定：E 对应战技，Q 对应爆发
  • 侧边栏和占位符可以显示当前能量、E 冷却和 Q 冷却。
  • 若技能没放出来，优先检查：当前角色、主手武器、冷却、能量、是否在线。

5. 反应怎么触发

这一节专门回答“不是只有名字，实际怎么打出来”。

蒸发怎么打

先让目标身上有 火，再用 水 打它，或者先挂 水 再用 火 打它。两种都会出蒸发，但当前源码里水打火更强、消耗也更高。

融化怎么打

先挂 冰 再上 火，或先挂 火 再上 冰。两种都会融化，但火打冰更强。

超载怎么打

让目标身上同时发生 火雷组合。常见做法是火角色先挂火，再让雷角色补刀，或反过来。

感电怎么打

先让目标带 水，再上 雷；或者先雷后水。触发后目标会进入持续跳电状态。

冻结与碎冰怎么打

先让目标同时吃到 水 + 冰，它会冻结。之后在本插件当前实现里，用 物理伤害 或 岩元素 再打它，就可能触发碎冰。

超导怎么打

让目标身上先有 冰 再上 雷，或者先有 雷 再上 冰。触发后会附带范围物理减抗。

原激化 / 超激化 / 蔓激化怎么打

先用 雷 + 草 把目标打进原激化。之后继续用 雷 打，会走超激化；继续用 草 打，会走蔓激化。

绽放 / 超绽放 / 烈绽放怎么打

先用 水 + 草 生成草原核。然后再让草原核吃到 雷 变超绽放，或吃到 火 变烈绽放。

燃烧怎么打

让 火 和 草 发生反应。触发后不是一下就结束，而是进入持续燃烧状态。

扩散怎么打

先让目标身上带 火 / 水 / 雷 / 冰 之一，再用 风 打。风本身不是直接扩散所有元素，而是扩散目标身上当前主导的可扩散元素。

结晶怎么打

先让目标身上带 火 / 水 / 雷 / 冰 之一，再用 岩 打。会掉落结晶碎片，拾取后给护盾。

为什么有时不是你想要的反应

因为源码会根据目标身上的附着量、冻结状态、感电双载体、激化状态和优先级来规划下一步反应，而不是单纯只看最后两次命中。

6. 面板与显示

  • 你可以用 /ys sidebar toggle 切自己的侧边栏。
  • 你可以用 /ys damage toggle 切自己的伤害显示。
  • 如果服里装了 PlaceholderAPI，这些数据还能接到外部菜单和计分板。
