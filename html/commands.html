Operations & UI

指令与 GUI玩家操作、服主管理、权限节点与管理界面

当前插件的控制面并不弱。除了完整的命令体系外，角色编队 GUI、命座 GUI、管理员 GUI 也已经具备实用价值。

别名

主命令为 /yuanshen，别名为 /ys。

Commands

主要命令

下面列的是当前源码中明确存在的命令入口。

重载配置

/ys reload
  • 重载主配置、角色配置、属性配置、侧边栏配置。
  • 重新加载角色、武器、怪物抗性定义。
  • 刷新在线玩家属性和侧边栏。

查看属性

/ys stats [玩家]
  • 查看当前角色、武器、元素和核心面板数值。
  • 可按权限读取自己或其他玩家数据。

伤害显示

/ys damage <toggle|on|off> [玩家]
  • 切换自己或目标玩家的伤害显示。
  • 设置结果会存入玩家数据文件。

侧边栏显示

/ys sidebar <toggle|on|off> [玩家]
  • 切换自己或目标玩家的侧边栏显示。
  • 会触发实时记分板刷新或卸载。

技能施放

/ys skill <e|q>
  • 通过命令直接释放战技或爆发。
  • 会检查角色、武器匹配、冷却与能量。

打开角色编队

/ys open
  • 打开角色编队 GUI。
  • 这里只存角色，不存武器。

查看可发放列表

/ys list <character|weapon>
  • 查看当前已加载的角色或武器定义。

发放内容

/ys give character <角色> [等级] [命座] [玩家] /ys give weapon <武器> [等级] [精炼] [玩家] /ys give constellation [数量] [玩家]

切换出战槽位

/ys player set <1|2|3|4>
  • 切换当前角色槽位并保存。

管理员 GUI

/ys admin
  • 打开腐竹管理 GUI。

元素快捷设置

/ys <火|水|冰|雷|风|岩|草|清除>
  • 设置当前元素并同步到玩家数据。

Interfaces

GUI 结构

玩家操作和服主管理都不只靠命令行。

角色编队 GUI

  • 27 格布局，含 4 个角色槽与 4 个指示槽。
  • 显示当前槽位、角色、主手武器状态、所需武器类型。
  • 可通过配置文件重定义布局和材质。

命座 GUI

  • 显示当前命座、下一命效果和命星数量。
  • 没有角色或命星不足时会阻止升级。

管理员 GUI

  • 在线玩家列表、玩家详情、武器发放和配置重载。
  • 还能切侧边栏、切伤害显示、清元素、刷新属性。
